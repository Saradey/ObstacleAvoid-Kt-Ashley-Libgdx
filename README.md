# ObstacleAvoidKotlin-GameLibgdx

- Kotlin
- Libgdx
- Scene2d
- Actor
- ESC - Entity Component System

## Hotkey:
- a - left move
-  d - right move
- input touch - left or right move
- z - zoom out camera
- x - zoom in camera
- c - show debug grid
- r - reset camera position
- space - game pause
- v - drawing sprite or not
- b - draw fps monitor, or not
